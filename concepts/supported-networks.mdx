ProofBridge is currently deployed on testnet networks. You can bridge assets between Ethereum Sepolia, Hedera Testnet, and Stellar Testnet using the [ProofBridge app](https://proof-bridge.vercel.app/) today. Mainnet deployments and additional EVM chains are on the roadmap.

<Warning>
  ProofBridge is **testnet-only** at this time. Do not send mainnet assets to any of the contract addresses below. Mainnet deployment is on the roadmap.
</Warning>

## Active cross-chain routes

<Tabs>
  <Tab title="ETH ↔ wETH">
    Bridge native ETH from Ethereum Sepolia to wrapped ETH on Hedera, or in the reverse direction.

    | Direction | Source token | Destination token |
    |---|---|---|
    | Sepolia → Hedera | ETH (auto-wrapped to WETH) | wETH |
    | Hedera → Sepolia | wETH | ETH (auto-unwrapped) |

    Native ETH is automatically wrapped to WETH on deposit and unwrapped back to ETH on withdrawal. You never need to manually wrap or unwrap.
  </Tab>
  <Tab title="HBAR ↔ wHBAR">
    Bridge native HBAR from Hedera Testnet to wrapped HBAR on Ethereum Sepolia, or in the reverse direction.

    | Direction | Source token | Destination token |
    |---|---|---|
    | Hedera → Sepolia | HBAR (auto-wrapped to WHBAR) | wHBAR |
    | Sepolia → Hedera | wHBAR | HBAR (auto-unwrapped) |

    The same automatic wrap/unwrap behavior applies to HBAR.
  </Tab>
  <Tab title="ProofBridge Token">
    An ERC20 example token deployed on both Sepolia and Hedera, demonstrating cross-chain ERC20 bridging.

    | Direction | Source token | Destination token |
    |---|---|---|
    | Sepolia → Hedera | ProofBridge Token (Sepolia) | ProofBridge Token (Hedera) |
    | Hedera → Sepolia | ProofBridge Token (Hedera) | ProofBridge Token (Sepolia) |

    This route is primarily for testing and developer integration. It shows how any ERC20 token pair can be configured on ProofBridge.
  </Tab>
</Tabs>

<Info>
  Adding a new EVM chain requires deploying the four contracts (`Verifier`, `MerkleManager`, `AdManager`, `OrderPortal`), configuring bidirectional chain links, and registering token routes. See the [smart contracts reference](/reference/smart-contracts) for details.
</Info>

## No registration required

You connect with your wallet — there are no accounts, usernames, or KYC requirements. Your wallet address is your identity on ProofBridge, and all trade terms are enforced on-chain rather than by any off-chain service.
